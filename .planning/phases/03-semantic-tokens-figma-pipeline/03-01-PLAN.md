**Step 1: Refactor style-dictionary.config.mjs to multi-instance loop (TOKEN-10)**

Replace the current single-instance config with the multi-instance pattern from 03-RESEARCH.md Pattern 2. The new config must have:

1. Keep the existing imports at top:
```javascript
import StyleDictionary from 'style-dictionary';
import { register, expandTypesMap } from '@tokens-studio/sd-transforms';
register(StyleDictionary);
```

2. Define brand/mode arrays:
```javascript
const BRANDS = ['parent-brand', 'child-brand'];
const MODES = ['light', 'dark'];
```

3. **Primitives instance (unchanged from Phase 2):** Keep the existing `sdPrimitives` instance that:
   - `source: ['tokens/primitives/**/*.tokens.json']`
   - `preprocessors: ['tokens-studio']`
   - `expand: { typesMap: expandTypesMap }`
   - CSS platform: `transformGroup: 'tokens-studio'`, `transforms: ['name/kebab']`, `prefix: 'dsx'`, `buildPath: 'dist/css/'`, destination `'tokens.css'`, format `'css/variables'`, `outputReferences: true`, `selector: ':root'`, filter excluding breakpoints
   - TS platform: breakpoints only to `src/breakpoints.ts`

4. **Semantic instances (NEW -- nested loop):** For each brand x mode combination:
   - `include: ['tokens/primitives/**/*.tokens.json']` -- loaded for alias resolution, NOT output
   - `source: ['tokens/semantic/${brand}/${mode}.tokens.json']` -- ONLY semantic tokens output
   - `preprocessors: ['tokens-studio']`
   - `expand: { typesMap: expandTypesMap }`
   - CSS platform only (no TS platform for semantics):
     - `transformGroup: 'tokens-studio'`
     - `transforms: ['name/kebab']`
     - `prefix: 'dsx'`
     - `buildPath: 'dist/${brand}/'`
     - destination: `'${mode}.css'`
     - format: `'css/variables'`
     - `outputReferences: true`
     - selector: `mode === 'light' ? ':root' : '[data-theme="dark"]'`

5. Execution order -- sequential per instance, clean before build:
```javascript
// Primitives first
await sdPrimitives.cleanAllPlatforms();
await sdPrimitives.buildAllPlatforms();

// Then semantic instances
for (const brand of BRANDS) {
  for (const mode of MODES) {
    // ... create SD instance ...
    await sd.cleanAllPlatforms();
    await sd.buildAllPlatforms();
  }
}
```

CRITICAL: Do NOT use `Promise.all()` for the loop -- sequential execution prevents buildPath overlap issues (per Research Pitfall 2).
