**1. SemanticColorTable.tsx** (`apps/storybook/stories/components/SemanticColorTable.tsx`)

A theme-reactive component that reads Storybook globals and displays semantic color tokens for the active brand/mode.

```typescript
import React from 'react'
import { useGlobals } from '@storybook/preview-api'
// Fallback: if @storybook/preview-api does not resolve, try 'storybook/preview-api'
import { TokenTable } from './TokenTable'
import { getSemanticTokensForTheme } from './token-data'
```

Implementation:
- Call `useGlobals()` to get `[globals]`
- Extract `brand` and `mode` from globals (default 'parent' and 'light')
- Call `getSemanticTokensForTheme(brand, mode)` to get semantic token rows
- Filter to only color-related tokens (name starts with `color.`)
- Group by semantic category: background, surface, text, link, action, focus, border, divider, icon, status, skeleton, highlight
- Render each group with a sub-heading (e.g. "Background", "Surface", "Text") using h4 elements
- Pass each group to `<TokenTable tokens={group} showAliasChain={true} renderPreview={(token) => <div style={{ width: 24, height: 24, borderRadius: 4, border: '1px solid var(--dsx-color-border-default)', backgroundColor: `var(${token.cssVar})` }} />} />`
- The color swatch preview uses `var(${token.cssVar})` as backgroundColor — this live-resolves to the current theme's value

If `useGlobals` import fails at build time, fall back to accepting `brand` and `mode` as props:
```typescript
interface SemanticColorTableProps {
  brand?: string
  mode?: string
}
```

**1. SpacingBar.tsx** (`apps/storybook/stories/components/SpacingBar.tsx`)

Renders a single horizontal bar whose width is proportional to the spacing value, with token details alongside.

```typescript
import React from 'react'
import { CopyToClipboard } from './CopyToClipboard'

interface SpacingBarProps {
  name: string       // e.g. "spacing.4"
  cssVar: string     // e.g. "--dsx-spacing-4"
  rawValue: string   // e.g. "16px"
}
```

Layout: horizontal row.
- Left side (120px fixed): token label — `name` in monospace, `rawValue` next to it in muted color. Both wrapped in CopyToClipboard.
- Right side: horizontal bar with:
  - `height: 16px` (fixed bar height for all bars — the bar represents spacing as WIDTH not height, per UI spec correction: "horizontal bar whose width = the spacing value")
  - `width: var(${cssVar})` — uses the CSS custom property directly so the bar width is the actual spacing value in pixels
  - `backgroundColor: var(--dsx-color-action-default)` (teal accent)
  - `borderRadius: 2px`
  - Max width constraint: `maxWidth: 100%` to prevent overflow on large values (spacing-96 = 384px)

For the CSS var name label: `fontSize: 11px`, `fontFamily: monospace`, `color: var(--dsx-color-text-muted)`.

Row styling: `display: flex`, `alignItems: center`, `gap: 8px`, `padding: 4px 0`, `borderBottom: 1px solid var(--dsx-color-border-subtle)`.
