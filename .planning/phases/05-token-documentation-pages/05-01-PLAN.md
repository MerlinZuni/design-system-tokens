In `packages/tokens/style-dictionary.config.mjs`, change line 90 (inside the `for (const brand of BRANDS)` / `for (const mode of MODES)` loop):

**BEFORE:**
```javascript
const selector = mode === 'light' ? ':root' : '[data-theme="dark"]';
```

**AFTER:**
```javascript
const selector = `[data-brand="${brand.replace('-brand', '')}"][data-theme="${mode}"]`;
```

This produces:
- `[data-brand="parent"][data-theme="light"]` for parent-brand/light
- `[data-brand="parent"][data-theme="dark"]` for parent-brand/dark
- `[data-brand="child"][data-theme="light"]` for child-brand/light
- `[data-brand="child"][data-theme="dark"]` for child-brand/dark

The brand constant values are `parent-brand` and `child-brand` — strip the `-brand` suffix so the data attribute values are `parent` and `child` (per D-01).

After editing, rebuild tokens:

2. **Create withThemeAttributes decorator** — sets data-brand and data-theme on document.documentElement:
```typescript
const withThemeAttributes: Decorator = (Story, context) => {
  const { brand = 'parent', mode = 'light' } = context.globals

  useEffect(() => {
    document.documentElement.setAttribute('data-brand', brand)
    document.documentElement.setAttribute('data-theme', mode)
  }, [brand, mode])

  return (
    <div style={{ background: 'var(--dsx-color-background-default)', padding: '2rem' }}>
      <Story />
    </div>
  )
}
```

3. **Add globalTypes for toolbar dropdowns** (per D-03):
