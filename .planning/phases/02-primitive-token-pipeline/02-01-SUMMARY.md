## Decisions Made

- Used `style-dictionary@^4.x` not v5 — sd-transforms v1.x targets SD v4; v5 would break the preprocessor
- `dependsOn: ["^build", "build:tokens"]` — no `^` on `build:tokens` because this is intra-package ordering (same package, not upstream packages)
- Removed `clean: true` entirely (not replaced with `clean: false`) — the property is gone
- `outputReferences: true` is pre-configured for Phase 3 semantic token aliasing via CSS var() chains
- Breakpoints filtered to TS-only platform because CSS var() cannot be used inside `@media` queries
