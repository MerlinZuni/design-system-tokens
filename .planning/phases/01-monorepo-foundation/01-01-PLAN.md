    2. Create `package.json` at root with EXACTLY this content:
    ```json
    {
      "name": "design-system-x",
      "version": "0.0.0",
      "private": true,
      "workspaces": [
        "packages/*",
        "apps/*"
      ],
      "scripts": {
        "build": "turbo run build",
        "dev": "turbo run dev",
        "lint": "turbo run lint",
        "test": "turbo run test",
        "changeset": "changeset",
        "version-packages": "changeset version",
        "release": "turbo run build && changeset publish"
      }
    }
    ```

    3. Create `turbo.json` with the v2 `tasks` key (NOT `pipeline`):
    ```json
    {
      "$schema": "https://turbo.build/schema.json",
      "tasks": {
        "build": {
          "dependsOn": ["^build"],
          "outputs": ["dist/**"],
          "inputs": ["src/**", "tsup.config.ts", "tsconfig.json", "package.json"]
        },
        "dev": {
          "dependsOn": ["^build"],
          "cache": false,
          "persistent": true
        },
        "lint": {
          "dependsOn": ["^lint"]
        },
        "test": {
          "dependsOn": ["^build"],
          "outputs": ["coverage/**"]
        }
      }
    }
    ```

    4. Create `tsconfig.base.json` with strict mode. CRITICAL: `composite` MUST be `false` (tsup uses esbuild, not tsc --build):
    ```json
    {
      "compilerOptions": {
        "target": "ES2020",
        "module": "ESNext",
        "moduleResolution": "Bundler",
        "lib": ["ES2020", "DOM"],
        "strict": true,
        "exactOptionalPropertyTypes": true,
        "noUncheckedIndexedAccess": true,
        "jsx": "react-jsx",
        "declaration": true,
        "declarationMap": true,
        "sourceMap": true,
        "esModuleInterop": true,
        "skipLibCheck": true,
        "isolatedModules": true,
        "composite": false
      }
    }
    ```

    5. Create `.gitignore` with:
    ```
    node_modules/
    dist/
    .turbo/
    storybook-static/
    *.tsbuildinfo
    .DS_Store
    ```

    6. Create directories: `packages/tokens/src/`, `packages/primitives/src/`, `apps/`

    7. Install root devDependencies:
    ```bash
    npm install --save-dev turbo typescript
    ```
    Note: ESLint and Changesets are installed in Plan 02 to keep this task focused on the build foundation.
  </action>
