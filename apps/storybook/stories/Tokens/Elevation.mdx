# Elevation

Elevation tokens define box-shadow values at multiple intensity levels. Style Dictionary decomposes composite shadow tokens into individual CSS custom properties per layer.

## Primitive Tokens

    <ElevationCard key={t.name} token={t} />
  ))}
</div>
