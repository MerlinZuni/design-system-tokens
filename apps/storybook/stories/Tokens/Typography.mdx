# Typography

The type scale provides 13 steps from `text-xs` (12px) to `text-9xl` (128px). All steps use the Archivo font family.

## Primitive Tokens

## Semantic Tokens

Typography does not currently have a semantic token layer. Typography tokens are primitives consumed directly. Semantic typography mappings (e.g., heading styles mapped to specific scale steps) are shown on the [Styles/Headings](?path=/docs/styles-headings--docs) page.
