# ColorSwatch

ColorSwatch displays a color token visually as a square swatch with a label.

## When to use

Use ColorSwatch in documentation pages to preview color tokens. It is a documentation primitive, not intended for production UI. Pass a CSS custom property reference via `tokenVar` and a human-readable `label` to give context. Use the `size` prop to fit different documentation grid layouts.
