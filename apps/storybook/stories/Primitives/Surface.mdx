# Surface

Surface renders a semantic container with background, border, and elevation from the design token system.

## When to use

Use Surface when you need a container whose visual properties adapt to the active brand and color mode. Pair `elevation` with `background` to create distinct visual layers. Use `background='card'` with `border={true}` for card-style surfaces, and `background='inverse'` for dark panels within a light-mode layout.
