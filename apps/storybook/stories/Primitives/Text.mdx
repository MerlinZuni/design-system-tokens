# Text

Text renders typographically styled text using the design system type scale.

## When to use

Use Text when you need to apply a design system type scale step to any text content. Pair with the `as` prop to render the semantically correct HTML element for accessibility and SEO. Choose `variant` to communicate visual hierarchy, and always use a semantic HTML element (`h1`–`h6`, `p`, `label`) when context demands it.

### Variant guidance

- `heading-lg` — Use with `as='h2'` for second-level section headings.
- `heading-md` — Use with `as='h3'` for third-level headings.
- `body-md` — Default variant. Use for all standard prose.
- `caption` — Use for supplemental metadata, timestamps, or helper text.
- `code` — Use for inline code fragments; renders in monospace.
