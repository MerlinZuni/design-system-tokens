# Stack

Stack arranges children vertically with consistent spacing from the design token scale.

## When to use

Use Stack for any vertical layout — form fields, card content, page sections. The `gap` prop maps to spacing tokens, ensuring consistent rhythm. Prefer Stack over raw flexbox with `flex-direction: column` so spacing is always sourced from the token system and reacts correctly to theme changes.
