# VisuallyHidden

VisuallyHidden renders content invisible to sighted users but accessible to screen readers.

## When to use

Use VisuallyHidden to provide screen reader labels for icon-only buttons, decorative images with context, or skip navigation links. Never use it to hide content that should be visible — it is specifically for supplemental screen-reader-only text that would be redundant or distracting to sighted users.
