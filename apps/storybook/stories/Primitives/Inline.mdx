# Inline

Inline arranges children horizontally with wrapping and consistent spacing.

## When to use

Use Inline for horizontal layouts like tag lists, button groups, or navigation items. Children wrap automatically when they exceed the container width. The `justify` prop controls main-axis distribution — use `justify='between'` to spread items across the full width of the container.
